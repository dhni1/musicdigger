# MUSICDIGGER UI Terms

MUSICDIGGER 화면에서 특정 위치를 쉽게 설명하기 위한 공통 용어집입니다.

## 기본 원칙

- 앞으로 화면을 설명할 때는 가능하면 이 문서의 용어를 우선 사용합니다.
- 같은 요소를 여러 이름으로 부르지 않고, 대표 이름 하나를 기준으로 말합니다.
- 괄호 안 표현은 비슷한 말로 이해해도 되는 별칭입니다.

## 전체 구조

- `앱 셸`: 화면 전체 바깥 구조. 왼쪽 사이드바와 오른쪽 메인 영역을 모두 포함합니다.
- `사이드바`: 화면 왼쪽 세로 메뉴 영역입니다.
- `메인 셸`: 사이드바 오른쪽의 실제 콘텐츠 영역입니다.
- `상단바`: 검색창, 브랜드, 프로필 버튼이 있는 맨 위 가로 바입니다.
- `페이지 영역`: 현재 선택한 화면(`Home`, `Map`, `Library`, `Profile`, `Settings`)이 나타나는 본문 영역입니다.
- `플레이어 바`: 화면 맨 아래쪽에 고정된 재생 바입니다.

## 사이드바 용어

- `기본 메뉴`: `Home`, `Map`, `Library`, `Playlists` 버튼 묶음입니다.
- `랜덤 버튼`: `Random Dig` 버튼입니다.
- `프로필 버튼`: 사이드바 맨 아래 `Profile` 버튼입니다.

## 상단바 용어

- `햄버거 버튼`: 왼쪽 위 3줄 메뉴 버튼입니다.
- `브랜드`: `MUSICDIGGER` 텍스트입니다.
- `검색창`: 장르 검색 입력창입니다.
- `프로필 슬롯`: 오른쪽 위 동그란 프로필 버튼입니다.
- `상단 메뉴 패널`: 햄버거 버튼을 눌렀을 때 나오는 작은 패널입니다.

## Home 화면 용어

- `홈 화면`: 기본 진입 화면입니다.
- `히어로`: 화면 상단의 큰 소개 영역입니다.
- `히어로 카피`: 히어로 왼쪽의 제목과 설명 영역입니다.
- `히어로 통계`: 히어로 오른쪽의 `Genres`, `Tracks`, `Connections` 카드 묶음입니다.
- `콘텐츠 그리드`: 히어로 아래 3단 구성 영역 전체입니다.
- `장르 라이브러리 패널`: 왼쪽 패널. 장르 카드들이 나오는 영역입니다.
- `장르 카드`: 장르 하나를 나타내는 카드 버튼입니다.
- `대표곡 패널`: 가운데 패널. 대표 트랙 목록이 나오는 영역입니다.
- `스포트라이트 카드`: 대표곡 패널 상단의 큰 선택 장르 요약 카드입니다.
- `트랙 리스트`: 대표곡이 줄 목록으로 나오는 영역입니다.
- `연결 패널`: 오른쪽 패널. 연결된 장르 묶음이 있는 영역입니다.
- `서브장르 블록`: `Subgenres` 영역입니다.
- `유사 장르 블록`: `Similar` 영역입니다.
- `퓨전 블록`: `Fusion` 영역입니다.

## Map 화면 용어

- `맵 화면`: `Map` 메뉴를 눌렀을 때 열리는 화면입니다.
- `맵 히어로`: 맵 화면 상단의 소개와 통계 영역입니다.
- `맵 통계`: `Visible`, `Connections`, `Mode` 카드 묶음입니다.
- `맵 그리드`: 맵 본체와 우측 설명 패널을 포함한 2단 영역입니다.
- `맵 패널`: 왼쪽의 실제 지도 영역이 있는 패널입니다.
- `맵 툴바`: 맵 패널 상단 오른쪽의 버튼 묶음입니다.
- `줌 컨트롤`: `-`, 퍼센트, `+`, `Reset` 버튼 묶음입니다.
- `풀맵 버튼`: `Open Full Map` 버튼입니다.
- `맵 뷰포트`: 화면에 실제로 보이는 맵 프레임입니다.
- `맵 서피스`: 뷰포트 안에서 더 크게 펼쳐진 실제 지도 캔버스입니다.
- `맵 노드`: 지도 위에 배치된 각 장르 이름 버튼입니다.
- `군집 라벨`: `Pulse`, `Spectrum`, `Distortion` 같은 흐릿한 구역 이름입니다.
- `연결선`: 현재 선택 장르와 연결 장르를 이어주는 선입니다.
- `맵 인스펙터`: 오른쪽 설명 패널 전체입니다.
- `선택 장르 제목`: 우측 패널 맨 위 장르 이름입니다.
- `선택 장르 설명`: 우측 패널 설명 문단입니다.
- `Nearby Links`: 우측 패널의 연결 장르 버튼 묶음입니다.
- `Major Tracks 미리보기`: 우측 패널의 대표곡 미리보기 목록입니다.
- `Open In Home 버튼`: 현재 장르를 홈 화면 기준으로 이어서 보게 하는 버튼입니다.
- `풀맵 모달`: 맵만 크게 띄워서 보는 팝업 화면입니다.
- `모달 맵`: 풀맵 모달 안의 큰 지도입니다.

## Library 화면 용어

- `라이브러리 화면`: `Library` 메뉴 화면입니다.
- `Spotify 연결 버튼`: `Connect Spotify` 또는 `Refresh Library` 버튼입니다.
- `프로필 카드`: Spotify 프로필 정보가 보이는 영역입니다.
- `플레이리스트 목록`: Spotify 플레이리스트 카드들이 나오는 영역입니다.
- `좋아요 트랙 목록`: Spotify 좋아요 트랙이 나오는 영역입니다.

## Profile 화면 용어

- `프로필 화면`: `Profile` 화면입니다.
- `프로필 히어로`: 프로필 화면 상단 소개 영역입니다.
- `계정 개요 블록`: Spotify 계정 상태를 보여주는 카드입니다.
- `라이브러리 요약 블록`: 플레이리스트 수, 좋아요 곡 수 등이 보이는 카드입니다.
- `프로필 설정 블록`: 테마 전환과 설정 이동 버튼이 있는 카드입니다.

## Settings 화면 용어

- `설정 화면`: `Settings` 화면입니다.
- `테마 블록`: 다크/라이트 모드를 바꾸는 카드입니다.
- `빠른 이동 블록`: `Back To Profile`, `Back To Home` 버튼이 있는 카드입니다.

## 모달 용어

- `플레이리스트 모달`: 플레이리스트 생성 팝업입니다.
- `맵 모달`: `Open Full Map`으로 여는 맵 전용 팝업입니다.
- `모달 배경`: 모달 뒤 어두운 반투명 배경입니다.

## 자주 쓰는 표현 예시

- "사이드바 `Map` 메뉴"
- "홈 화면 히어로 오른쪽 통계 카드"
- "장르 라이브러리 패널의 첫 번째 장르 카드"
- "맵 패널 상단 맵 툴바"
- "맵 뷰포트 안의 장르 노드"
- "맵 인스펙터의 `Nearby Links` 영역"
- "풀맵 모달의 줌 컨트롤"

## 메모

- 새 UI가 생기면 이 문서에 용어를 추가합니다.
- 이름이 애매한 요소는 먼저 이 문서 기준으로 이름을 정한 뒤 작업합니다.
